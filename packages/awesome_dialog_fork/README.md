# awesome_dialog (Cookster fork)

Vendored from [awesome_dialog](https://pub.dev/packages/awesome_dialog) 3.3.0 with **Rive removed**. Dialog headers use **Material `Icon`s** instead of `.riv` assets, so the app does not depend on `rive` / `rive_native` (avoids iOS native linker issues with prebuilt Rive static libraries).

Keep in sync with upstream only if you need new `awesome_dialog` features; re-apply the header change to stay Rive-free.
